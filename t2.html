TEMA BESKRIVELSE

I Tema 2 lærte vi om forskellige layoutprincipper: mobile-first design, responsive design, og CSS grids. Jeg valgte at bruge mobile-first princippet til at løse mobilsite-opgaven, da det fokuserer på de mest nødvendige elementer først.

MOBILSITE

Fokus, rolle og løsning

Mit Fokus

Jeg udviklede den første version af mit website med mobile-first princippet:

  • Designede grundstrukturen med undersider og navigationsmenu
  • Fulgte de udleverede wireframes og layoutdiagrammer

Min Rolle

  • Implementering og kodning af mobilversionen fra bunden HTML-struktur
  • CSS-styling, og validering af websitet

Min Løsning

  • Analyse af wireframes og layoutdiagrammer
  • Oprettelse af HTML-struktur
  • Tilføjelse af elementer: header, nav, main, footer
  • CSS styling for margin og padding, to forskellige fonte og farver.
  • Sikring af korrekt validering . og indsættelse af meta-tag

WEBSITE

Fokus, rolle og løsning

Mit Fokus

Jeg videreudviklede mit mobilsite, så det fungerer optimalt på desktops ved at bruge:

  • Media Queries
  • CSS Grids

Min Rolle

  • Tilpasse layoutet til større skærme
  • Uploade til mit domæne

Min Løsning

  • Analyse: Identifikation af ændringer for større skærme.
  • Implementering: Media queries og CSS grids.
  • Test: Design og layout på forskellige skærmstørrelser.
  • Upload: Validering og upload til mit domæne.
